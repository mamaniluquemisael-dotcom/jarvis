# Guía de Contribución

## Cómo Contribuir a JARVIS

### 1. Fork el Repositorio
```bash
# Haz clic en "Fork" en GitHub
```

### 2. Clona tu Fork
```bash
git clone https://github.com/tu-usuario/jarvis.git
cd jarvis
```

### 3. Crea una Rama
```bash
git checkout -b feature/tu-feature
```

### 4. Haz Cambios y Commit
```bash
git add .
git commit -m "Add nueva funcionalidad"
```

### 5. Push a tu Fork
```bash
git push origin feature/tu-feature
```

### 6. Abre un Pull Request
- Ve a GitHub
- Click en "Pull requests"
- Click en "New Pull Request"
- Describe tus cambios

## Pautas

### Código
- Usa Python 3.8+
- Sigue PEP 8
- Añade comentarios en código complejo
- Prueba tu código antes de pushear

### Commits
- Mensajes claros en inglés o español
- "Add feature X"
- "Fix bug in Y"
- "Update documentation"

### Pull Requests
- Describe qué cambias
- Explica por qué
- Enlaza issues relacionados

## Ideas para Contribuir

- 🐛 Reportar bugs
- ✨ Sugerir nuevas funcionalidades
- 📝 Mejorar documentación
- 🌍 Traducir a otros idiomas
- 🧪 Escribir tests
- 🎨 Mejorar UI/UX
